# Objective
Baseline CNN for Image-to-Image Translation (Encoder–Decoder without GAN)

## To implement a baseline encoder–decoder CNN for paired image-to-image translation and analyze its performance using reconstruction loss.

## Experiment Task:
 1. Load paired images
2. Normalize to [-1, 1]
3. Train encoder–decoder CNN
4. Compute MSE / L1 loss
5. Visualize translated images

## Dataset
CIFAR10

## Expected Output
Output images are blurry
